# Desafio-de-Ingenieria-de-Datos-Covalto
Repositorio del Desafío de Ingeniería de Datos de Covalto. Contiene la resolución documentada de problemas de alto nivel (arquitectura y diseño de soluciones de datos para banca), bajo nivel (implementación y pruebas de algoritmos sobre DAGs) y escenarios prácticos con mocks. Incluye análisis, visualizaciones y estrategias colaborativas.
